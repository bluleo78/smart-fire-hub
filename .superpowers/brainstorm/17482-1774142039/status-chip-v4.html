💡 전체 인터랙션 정리

동작 결과 칩 변화
보기 아이콘+색상으로 모드/상태 즉시 파악 —
호버 상태 카드 드롭다운 (세션, 대화 수, 토큰) 밝기 증가
클릭 1회 닫힘 → 사이드 패널 오픈 ✦ + 분할 아이콘 + 대화 수
클릭 2회 사이드 → 풀스크린 전환 ✦ + 채움 아이콘 + 대화 수 (보라 배경)
클릭 3회 풀스크린 → 닫힘 ✦ + 녹색 점 (원래 상태)
⌘K 어디서든 사이드 패널 토글 해당 모드에 맞게 변경
