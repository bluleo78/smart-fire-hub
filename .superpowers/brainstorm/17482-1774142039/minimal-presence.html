최소 공간 + 항상 보이는 AI 진입점

공간은 거의 차지하지 않으면서 존재감을 유지하는 방법들

A

Edge Tab (화면 가장자리 탭)

화면 우측 또는 하단 가장자리에 붙은 반원형 탭. 클릭하면 사이드 패널이 열림. 3~4px만 차지.

우측 가장자리 AI 탭
📊 서울시 화재통계 (2,340건)
📊 소방서 출동현황 (5,120건)
📊 건축물 안전등급 (890건)
✦ AI

차지 공간: ~28px 폭. 콘텐츠를 가리지 않음. 단, 시각적 임팩트가 약할 수 있음.

B

Smart Mini FAB (컨텍스트 인식 미니 버튼)

기존 FAB보다 훨씬 작은 미니 아이콘(36px)이 우하단에 위치. 스크롤/페이지 전환에 따라 위치가 스마트하게 조정되어 콘텐츠를 가리지 않음. 펄스 애니메이션으로 생동감.

스마트 미니 FAB — 평상시
📊 서울시 화재통계 (2,340건)
📊 소방서 출동현황 (5,120건)
📊 건축물 안전등급 (890건)
✦

차지 공간: 36x36px. 이전 FAB 문제 해결: 크기 축소 + 콘텐츠 감지로 위치 자동 조정.

C

Header AI Chip (헤더 내 AI 칩)

메인 콘텐츠 영역의 페이지 헤더/브레드크럼 옆에 AI 칩 배지를 배치. 기존 헤더 공간을 활용하므로 추가 영역 불필요. 클릭하면 사이드 패널 오픈.

페이지 헤더에 AI 칩
데이터셋 › 서울시 화재통계
✦ AI ⌘K
⚙
📊 서울시 화재통계 (2,340건)
📊 소방서 출동현황 (5,120건)
📊 건축물 안전등급 (890건)

차지 공간: 0px 추가 (기존 헤더 활용). 가장 절약적이지만 존재감이 상대적으로 약할 수 있음.

D

Glow Strip (하단 빛나는 라인)

화면 최하단에 2~3px 높이의 그라데이션 라인. AI가 대기 중임을 암시하는 은은한 빛 효과. 클릭하면 입력 바가 위로 올라오고, 다시 클릭하면 라인으로 축소.

하단 Glow Strip — 대기 상태
📊 서울시 화재통계 (2,340건)
📊 소방서 출동현황 (5,120건)
📊 건축물 안전등급 (890건)
클릭 후 — 입력 바 확장
📊 건축물 안전등급 (890건)
✦
AI에게 질문하세요...
ESC로 닫기

차지 공간: 3px. 존재감은 빛으로 표현. 클릭하면 입력 바로 확장 → 전송하면 사이드 패널 오픈.
